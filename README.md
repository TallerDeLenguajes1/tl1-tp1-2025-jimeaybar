#### 2) Es conveniente incluir .gitignore porque actúa como un filtro que evita ensuciar el repositorio. Evita subir archivos pesador o innecesarios. Impide que subas por error archivos con contraseña o configuraciones sensibles. Mantiene el historial de Git enfocado solo en cambios de código real.
#### Se debe crear al principio del proyecto ya que solo ignora archivos que no han sido rastreados todavia.
#### El archivo .gitignore es un archivo de texto simple ubicado en la raíz del repositorio. Se configura escribiendo patrones de nombres de archivos o carpetas.
 - nombre_archivo.ext: ignora un archivo especifico
- carpeta/ : ignora totod el contenido de una carpeta.
- *.log: ignora todos los archivos que terminen en esa extensión.
#### Para ignorar el archivo ignorado.txt debo primeramente crear la carpeta .gitignore en mi repositorio (touch .gitignore) y luego escribir este comando 'echo ignorado.txt >> .gitignore'
